# DHT2xProtocolAnalyzer
DHT22 Temp/Humidity Sensor Protocol for Kingst Logic Analyzer using KingskVIS software
# Screenshot
![Overview Screenshot](/images/Screenshot1.png)
# Description
You will need to download the [KingstVIS Analyzer SDK](http://www.qdkingst.com/en/download) and build the source by following the instructions provided with the SDK.
The code is based on a protocol anaylyzer written for DHT2x Protocol Analyzer for Saleae Logic found here: [jakeson21/DHT2xProtocolAnalyzer](https://github.com/jakeson21/DHT2xProtocolAnalyzer)
